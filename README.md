# Task-Scheduler

A simple task scheduling program in C:

# Overview  
Task-Scheduler is a C program that lets you create and manage tasks, then automatically handles their execution according to scheduling logic.  
It demonstrates basic concepts of scheduling, time management, and usage of C language features — useful for learning confusing concepts of or for academic practice.  

# Features  
- Add new tasks with their parameters (task name, duration, focus level)  
- Schedule tasks in an order according to scheduling algorithm (it is based on focus level priority)  
- Display scheduled order and execution result (displays a 24 hour task arrangeement with timestamp based on input priority level)
  
# Folder Structure  
- MY_CODE.c — main source file containing scheduler implementation
